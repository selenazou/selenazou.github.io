Project Abstract

ScummVM is a program which provides support for users to play select adventure or role-playing games on systems for which they were not designed. It is an open-source project. Further details can be found on the project website: scummvm.org.

Over the course of this project, we will analyze ScummVM's documentation and implementation to understand its conceptual and concrete architecture.
